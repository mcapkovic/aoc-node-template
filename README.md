# aoc-node-template

aoc-node-template is a node js template for advent of code

## Installation

clone the repo and run 

```bash
npm install
```

## Usage

```bash
npm run day 1 1 # run day 1 part 1
```

*optional - create .env file in the root

```js
SESSION="" // your aoc session
YEAR="2017" // aoc year
```
